### vue 2 和 vue3 的区别
· vue 3 的template 支持多个根标签，2 不支持
· vue 3 有createApp() ，2是new Vue()
· vue 3接受组件 createApp(组件) ，vue 2  接受对象 new Vue({ template, render })
